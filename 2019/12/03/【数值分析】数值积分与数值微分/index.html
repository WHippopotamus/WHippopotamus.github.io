【数值分析】数值积分与数值微分

一、数值积分

插值型求积公式

类型 n m 求积公式：$\int_{a}^{b}{f(x)}dx\approx$ 截断误差
梯形 1 2 $\frac{b-a}{2}[f(a)+f(b)]$ $-\frac{(b-a)^3}{12} f^{“}(\eta)$
Simpson 2 3 $\frac{b-a}{6}[f(a)+4f(\frac{a+b}{2})+f(b)]$ $-\frac{(b-a)^5}{2880} f^{(4)}(\eta)$
Cotes 4 5 $\frac{b-a}{90}[7f(a)+32f(a+h) +12f(a+2h)+32f(a+3h)+7f(b)]$ $-\frac{2(b-a)}{945} (\frac{b-a}{4})^6 f^{(6)}(\eta)$

Cotes系数

n $c^{n}_i$
1 $[1\ \ \ 1]/2$
2 $[1\ \ \ 4 \ \ \ 1]/6$
3 $[1\ \ \ 3 \ \ \ 3 \ \ \ 1]/8$
4 $[7\ \ \ 32 \ \ \ 12 \ \ \ 32 \ \ \ 7]/90$

复化求积公式（区间数：$n$）

类型 节点 求积公式：$\int_{a}^{b}{f(x)}dx\approx$ 截断误差
复化梯形 n+1 $Tn(f)=\frac{h}{2}{[f(a)+2\sum{i=1}^{n-1}f(x_i)+f(b)]}$ $E_{T_n}(f)=-\frac{b-a}{12}h^2f^{“}(\eta)$
复化Simpson 2n+1 $Sn(f)=\frac{h}{6}{[f(a)+2\sum{i=1}^{n-1}f(xi)+4\sum{i=0}^{n-1}f(x_{i+\frac{1}{2}})+f(b)]}$ $E_{S_n}(f)=-\frac{b-a}{2880}h^4f^{(4)}(\eta)$
复化Cotes 3n+1 $E_{C_n}(f)=-\frac{2(b-a)}{945}(\frac{h}{4})^6f^{(6)}(\eta)$

误差的后验估计方法：区间逐次分半，自适应步长

类型 重要关系
复化梯形 $I(f)\approx T{2n}(f)+\frac{1}{3}[T{2n}(f)-T_{n}(f)]=S_n(f)$
复化Simpson $I(f)\approx S{2n}(f)+\frac{1}{15}[S{2n}(f)-S_{n}(f)]=C_n(f)$
复化Cotes $I(f)\approx C{2n}(f)+\frac{1}{63}[C{2n}(f)-C_{n}(f)]S_n(f)=R_n(f)$

计算截至条件：$|T{2n}(f)-T{n}(f)| \leq \varepsilon$

二、数值微分

插值型求导公式

  1. 两点数值微分公式
    • $ f’(x_0)=\frac{1}{h}[f(x_1)-f(x_0)]-\frac{h}{2}f^{“}(\xi_1)$
    • $ f’(x_1)=\frac{1}{h}[f(x_1)-f(x_0)]+\frac{h}{2}f^{“}(\xi_2)$
  2. 三点数值微分公式
    • $ f’(x_0)=\frac{1}{2h}[-3f(x_0)+4f(x_1)-f(x_2)]+\frac{h^2}{3}f^{(3)}(\xi_1)$
    • $ f’(x_1)=\frac{1}{2h}[f(x_2)-f(x_0)]-\frac{h^2}{6}f^{(3)}(\xi_2)$
    • $ f’(x_3)=\frac{1}{2h}[f(x_0)-4f(x_1)+3f(x_2)]+\frac{h^2}{3}f^{(3)}(\xi_3)$
