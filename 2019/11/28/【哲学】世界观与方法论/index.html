【哲学】世界观与方法论

科学的世界观与方法论

  • 科学的方法论：形式逻辑体系+实验验证，正是这两点使得科学成为了一门逻辑自洽的、精确的学问
  • 通过公理化定义就可以构建一个完备的体系
    • 欧氏几何
    • 黎曼几何
    • 范数理论
  • 对概念定理理解的三个层次：
    • 描述了什么？
    • 为什么是这样？
    • 有什么作用？
  • 对学习的认识
    • 学习的本质就是用有限的认识来构建可以应对无限可能的模型
    • 机器学习：数据--学习--模型--应用
    • 思维导图：线性思维\(\rightarrow\)网络思维、发散思维
    • 费曼技巧：学习+解释
    • 人类思维的生物学载体：生物神经网络
    • 学习时间不是衡量学习效果的标准，有效的成果产出才是

哲学的世界观与方法论

  • 认识论
    • 知己知彼，百战百胜
    • 未知是人类最大的恐惧来源，因此克服恐惧最好的方法就是面对恐惧
    • 压力感来自与我们不充分的准备
    • 见多识广
    • 人不可能想象出从来没有见过的东西，任何文艺形象的创作都能在现实中找到对应的原型。以人类对维度的认识为例，无论我们在数学上的论证过程如何缜密，但我们永远无法想象出真正的高维空间；反之，如果我们能够简单的体验一下高维空间，即使对数学一窍不通，我们对世界的认识也会上升一个很大的台阶。
  • 普遍联系与永恒发展
    • 把握事物间的区别与联系，是提升全面认识的关键
    • 近朱者赤，近墨者黑
    • 雪崩的时候没有一片雪花是无辜的
  • 世界的基本特性：
    • 因果性
    • 永恒运动
