【哲学】再来亿点方法论

Date: November 28, 2019  11:21:50 Category: Philosophy

一、科学的世界观与方法论

  • 科学的方法论：形式逻辑体系+实验验证，正是这两点使得科学成为了一门逻辑自洽的、精确的学问
  • 通过公理化定义就可以构建一个完备的体系
    1. 欧氏几何
    2. 黎曼几何
    3. 范数理论
  • 对概念定理理解的三个层次：
    1. 描述了什么？
    2. 为什么是这样？
    3. 有什么作用？
  • 对学习的认识
    1. 学习的本质就是用有限的认识来构建可以应对无限可能的模型
    2. 机器学习：数据--学习--模型--应用
    3. 思维导图：线性思维\(\rightarrow\)网络思维、发散思维
    4. 费曼技巧：学习+解释
    5. 人类思维的生物学载体：生物神经网络
    6. 学习时间不是衡量学习效果的标准，有效的成果产出才是
    7. 在遇到一个新的问题时，尽量将它往已知的问题上靠，并格外留意它们的相同点与不同点
  • 通解思维：在处理一些有共同点的问题时，为提高效率，最好的办法就是寻找处理这一类办法的通解，且必须满足如下条件：
    1. 通用性：适合处理绝大多数情况，能描述问题的本质
    2. 实用性：其形式最有利于处理生活中最常见的问题
    3. 形式简洁：便于记忆与传播
    4. 灵活性：能够根据不同的附加条件作出相应的变形，以处理各类特殊情况
  • 学会制定性能指标或指标体系
  • 学会起名字：通过命名可以将一类知识打包记忆，强行记住一个知识的所有细节实在太难了。通过名字，我们就可以快速联想到这个知识点包含的主要内容。起名字本身也是一门学问，要能够描述本质、方便好记、有文化内涵等等。只要勤加练习，终能成一代命名鬼才。
  • 结构化记忆

二、哲学的世界观与方法论

  • 认识论
    1. 知己知彼，百战百胜
    2. 未知是人类最大的恐惧来源，因此克服恐惧最好的方法就是面对恐惧
    3. 压力感来自与我们不充分的准备
    4. 见多识广
    5. 人不可能想象出从来没有见过的东西，任何文艺形象的创作都能在现实中找到对应的原型。以人类对维度的认识为例，无论我们在数学上的论证过程如何缜密，但我们永远无法想象出真正的高维空间；反之，如果我们能够简单的体验一下高维空间，即使对数学一窍不通，我们对世界的认识也会上升一个很大的台阶。
  • 普遍联系与永恒发展
    1. 把握事物间的区别与联系，是提升全面认识的关键
    2. 近朱者赤，近墨者黑
    3. 雪崩的时候没有一片雪花是无辜的
  • 世界的基本特性：
    1. 因果性
    2. 永恒运动

三、生活的方法论

  • 物以类聚，人与群分

四、自问自答

怎样提交计算能力？

  1. 格式清晰
  2. 适当放慢计算速度
  3. 勤加练习

如何有效的讲述一个故事？

  1. 记叙文的六个基本要素：时间、地点、任务；起因、经过、结果
  2. 锦上添花的技巧：幽默性、修饰、伏笔
